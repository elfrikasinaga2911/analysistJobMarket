# ANALYSIST JOB MARKET
TUGAS 3 BUSINESS INTELLIGENCE
Autor: Elfrika Sari M Sinaga_41823010076

Data
- Data studi kasus Analysing Job Market Data format CSV
- Data Cleaning ke Power BI
- Transformasi dengan penghapusan duplikat dan standarisasi nama kolom.

Fitur Utama
Jobs Dashboard
Menampilkan jumlah serta persebaran lowongan berdasarkan lokasi, level posisi, serta tren permintaan pekerjaan.

Skills Dashboard
Menggambarkan keterampilan yang paling banyak dicari di industri. Termasuk analisis hubungan keterampilan dengan gaji dan level pekerjaan.

Company Dashboard
Menunjukkan daftar perusahaan perekrut, ukuran perusahaan, hingga distribusi jumlah lowongan.

Analisis Tambahan
Mencakup analisis salary, job title, pengalaman, job level, skill likelihood, dan aspek terkait lainnya.
